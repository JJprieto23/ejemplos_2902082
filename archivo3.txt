# Archivo 3

Lenguajes de programación

- C++: Es un lenguaje de programación de propósito general ampliamente utilizado en el desarrollo de sistemas, juegos, software de alto rendimiento y aplicaciones de escritorio. Es una extensión del lenguaje C con características de programación orientada a objetos.

- C#: Desarrollado por Microsoft, es un lenguaje de programación que se utiliza principalmente para el desarrollo de aplicaciones de escritorio de Windows, aplicaciones web y juegos a través de la plataforma Unity.

- PHP: Es un lenguaje de programación ampliamente utilizado para el desarrollo de sitios web dinámicos y aplicaciones web. Es especialmente conocido por ser el lenguaje principal para la creación de sitios web basados en WordPress.
